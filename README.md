# Nexura Portfolio Website

This is the official portfolio website for **Nexura**, a digital engineering agency specializing in advanced software, AI, and Cloud solutions.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Fonts:** Montserrat (Headlines) & JetBrains Mono (Code/Technical)

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```

3.  **Open in Browser:**
    Navigate to [http://localhost:3000](http://localhost:3000)

## Design System

The project strictly follows the **Nexura Graphic Charter**:
- **Primary Color:** Electric Cyan (`#00D7D7`)
- **Background:** Midnight Navy (`#051221`)
- **Theme:** Cyberpunk / Deep Tech
